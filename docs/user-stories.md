# Conecta IFPE

## Histórias do Usuário

```
Como assistente social,
Eu gostaria de visualizar a lista de alunos aprovados para a bolsa,
Para poder monitorar e gerenciar os pagamentos de forma eficiente.
```

```
Como aluno,
Eu gostaria de acessar meu histórico de recebimento,
Para poder acompanhar os pagamentos que recebi ao longo do tempo.
```

```
Como departamento financeiro,
Eu gostaria de receber a lista de pagamento gerada pela assistente social,
Para poder realizar os pagamentos de maneira organizada e dentro do prazo.
```

```
Como assistente social,
Eu gostaria de receber notificações sobre os alunos que foram removidos da lista de pagamento,
Para que eu possa atualizar a lista de espera e informar os alunos impactados.
```

```
Como aluno,
Eu gostaria de ser notificado quando o pagamento da minha bolsa for realizado,
Para que eu esteja ciente e possa planejar meus gastos.
```

```
Como administrador,
Eu gostaria de gerenciar as permissões dos usuários no sistema,
Para garantir que cada perfil tenha acesso apenas às funcionalidades necessárias.
```

```
Como aluno,
Eu gostaria de saber o status da minha bolsa,
Para que eu possa entender se ela está ativa ou se houve algum problema.
```

```
Como departamento financeiro,
Eu gostaria de gerar relatórios sobre os pagamentos realizados,
Para que eu possa analisar a distribuição das bolsas.
```

```
Como aluno,
Eu gostaria de ter um canal de comunicação direto com a assistente social,
Para que eu possa tirar dúvidas rapidamente sobre a minha bolsa.
```

```
Como assistente social,
Eu gostaria de ter acesso ao extrato de pagamento gerado pelo departamento financeiro,
Para que eu possa verificar os pagamentos realizados e não realizados.
```

```
Como assistente social,
Eu gostaria de ter uma ordenação das inscrições por probabilidade de aceitação na bolsa,
Para que eu possa agilizar a análise das inscrições e priorizar aquelas com maior chance de aprovação.
```

```
Como aluno,
Eu gostaria de ter mais visibilidade sobre o status do meu pagamento, como quando está disponível para saque, quando caiu na conta ou se houve algum erro no pagamento,
Para que eu possa me programar melhor e ter maior tranquilidade em relação aos meus recursos financeiros.
```
